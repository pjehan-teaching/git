Branches

La commande suivante permet d’afficher la liste des branches disponibles sur un dépôt git :

git branch

Branches

Vous pouvez créer une nouvelle branche sur votre dépôt git et vous positionner sur cette nouvelle branche :

git branch develop
git checkout develop

Ou plus simplement :

git checkout -b develop
git branch -d develop permettra de supprimer la branche develop

Branches

Une fois que vous êtes satisfait des modifications apportées sur une branche, vous pouvez effectuer une opération de fusion :

git checkout master
git merge develop
