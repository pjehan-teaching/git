Hooks

Les hooks (ou crochets) sont des scripts pouvant être exécuté à différent moment en complément de git.

Exemples :

  • Côté client, afin de valider le message de commit.
  • Côté client, afin de valider la conformité du code (avec un outil comme lint par exemple).
  • Côté serveur, afin de déployer automatiquement le site internet lorsque l’on push sur la branche master.
  • ...
